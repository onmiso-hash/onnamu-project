Registration Data Access Protocol - Web Client

[KISA RDS] - Registration Directory Service

RDAP 조회

• 현재 페이지는 Client-side(JavaScript) Web Client 프로그램이며, Server-side Web Client 로의 전환은 [여기]를 클릭하시기 바랍니다.

• RDAP Client는 JavaScript를 사용하여 HTTP 상에서 인터넷 주소정보를 JSON 형식으로 조회 합니다.

• 아래의 Query Example을 선택하시거나 직접 대상 Object를 선택 후 주소 문자열을 입력하시여 조회(Query)를 실행해 보시기 바랍니다.

• 언어(language) 선택 값은 RDAP Request Headers(Accept-Language)에 설정되어 RDAP-Server로 전달됩니다.

Query Autnum : 9700 , 9711
